# simple_vm

A simple bytecode Virtual Machine.

Version: `0.0.1`

# Compilation

In order to compile, you will need g++ and make. 

Then, go to the root directory (`/simple_vm/`) and simply type:

```
make
```

The final program (`.exe`) will be located in `/bin/vm.exe`
